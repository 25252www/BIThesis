<h2>欢迎</h2>

BIThesis 是针对北理工毕业设计报告而制作的一个非官方的 Latex 模板，同时包括其他相关报告模板。根据北京理工大学教务部发布的毕业论文格式要求纯手工打造而成。

## 项目内容

本项目旨在维护本科阶段尽可能多的论文、实验报告，以及毕业设计论文等模板。目前该项目正在进行大量的维护，我们的目标是为您带来最可重用的 Latex 模板，您可以直接用于您的大部分工作。
